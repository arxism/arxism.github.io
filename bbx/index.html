Installation and Usage

A bookmarklet is a link that contains JavaScript. When a bookmarklet link is activated, the browser runs the JavaScript code that it contains.

Clicking/tapping on the links below will run the script here, for preview. They will update automatically once installed and you can switch from one to another by clicking on the script name in the bottom right of the resulting dialog. Each has a "help" tab with an explanation about how it works and the features it provides.

See below about adding the bookmark to your browser:

    To use on Desktop:

    1. Toggle the display of the bookmarks toolbar.
      - The shortcut for doing so in Chrome, Firefox and Safari is Shift + Ctrl/Cmd + B.
    2. Drag the bookmark link into your browser's bookmarks toolbar.

    To use on Mobile:

    1. Click the "copy" link next to the bookmarklet you want.
    2. Bookmark this page with your browser's bookmark.
      - usually an in-menu star or similar
    3. Click "edit" on the popup notification.
    4. Change the name of the bookmarklet to "bbx" or similar.
    5. Paste the contents of your clipboard into the "url" field.
    6. Click "save"
    7. When you want to run the script on a page,
      - type "bbx" into the url bar and tap the script
      - or find it in your bookmarks manager.
